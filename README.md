# <a name="Description"></a> ОПИСАНИЕ

Вы играете за маленького мальчика, который ищет путь из этого проклятого лабиринта. Сможете ли вы выйти из него живым преодолевая препятствия?

# <a name="Controls"></a> Управление

Ходьба:

W - движение вверх;
A - движение влево;
S - движение вниз;
D - движение вправо;
